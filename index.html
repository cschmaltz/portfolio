Project 01
AI Voice Agent: An Inbound Call Handling, Booking & Deposit Collection
View Build
Built a fully automated AI voice agent that answers every inbound call for a service business, qualifies the caller, collects booking details, and sends a secure Stripe payment link via SMS zero human involvement required.

The agent handles the entire conversation naturally using ElevenLabs conversational AI. Once booking details are collected, n8n triggers automatically: generates a Stripe payment link, sends it via Twilio SMS, and waits for payment confirmation. When the customer pays, Stripe webhooks back to n8n which books the appointment in Google Calendar, updates the tracking sheet, and sends a confirmation SMS.

Works for any service business that takes appointments and deposits salons, barbershops, HVAC, plumbers, law firms, medical clinics.
ElevenLabs n8n Stripe Twilio Google Calendar Google Sheets
Project 02
AI LinkedIn Lead Generation Scrape, Score, Enrich & Outreach Automatically
View Build
Built a 3-workflow LinkedIn outreach system that runs daily without manual input. Pulls qualified leads from LinkedIn via Apify, enriches each one with company data via RapidAPI and Tavily, scores lead quality using AI, finds verified emails via Prospeo, and generates personalised outreach messages based on each lead's actual LinkedIn activity.

Inbound DM replies route automatically through a Slack human-in-the-loop approval workflow. Approved replies send. Rejected replies loop back with feedback. When a lead books via cal.com, the system generates a pre-call summary email automatically.

Deployed for an AI automation agency generating 15–25 qualified leads per day at near-zero cost using free API tiers.
n8n Apify RapidAPI Tavily Prospeo Pinecone Gemini Slack Brevo

Project 05
RAG Pipeline PDF Ingestion, Vector Storage & AI Query System for Business Documents
View on Upwork
Built a Retrieval Augmented Generation pipeline that allows businesses to query their own documents in plain English. Ingests PDFs product catalogs, compliance documents, knowledge bases chunks them, generates embeddings via Gemini, and stores them in Pinecone.

Users ask questions in natural language. The system searches Pinecone for relevant context, passes it to Gemini, and returns accurate answers sourced from the business's own documents. No hallucinations answers are grounded in real business data. No custom model training required.

Built for professional services clients who needed AI that knows their specific products, policies, and processes.
